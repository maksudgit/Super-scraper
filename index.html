What is Super-scraper ?

Super-scraper is a browser plugin that will help you get data out of websites and turn potential into opportunities

Why Super-scraper

No coding skills required
Super-scraper has a point and click interface helping you select the data you want. Optionally you can set you're own CSS/Jquery selectors.

No sign up required
No need to provide an email or register. You can download the web scraper and start scraping immediately with 100 credit free upon install.

No monthley subscription
Unlike other web scraping services we think a monthley subscription is absolutely ridiculous. Instead you can purchase an API key that holds a certain amount of credit with no expiration date and use it whenever you need it.

No dynamic data issues
Web scrapers are notorious of failing miserably when it comes to handling dynamic data like
1. infinite scrolling websites
2. Javascript content
3. data behind a login
Because super-scraper runs inside the browser it solves all these problems without any additional hassle required.

Frequently asked questions

Every time you select something a call is made to the server in order to process/parse the selection. You will see a loading icon inside the application every time this happens indicating a call is being made.
The amount of table rows or "records" returned depends on "what" and "how" you are scraping. Therefore "1" API call can return 1, 10, 100 or any amount of table rows based on what you are web scraping.
If you decide to purchase credit you will recieve an API Key in the email address you provide during the checkout.
The API Key is used to "identify" the credit you have purchased and is a string of characters. For example"c54b515b". Watch the "How to install" video to understand where to enter the Key.
You can change the "quantity" field during the checkout. For example; a quantity of "5" will get you 5 X 10K API calls.
